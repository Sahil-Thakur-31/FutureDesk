# FutureDesk

FutureDesk is a cross-platform personal career management system built as a monorepo with:

- `backend`: Express + TypeScript + MongoDB + Socket.io
- `desktop`: Electron + React + Vite + TailwindCSS
- `mobile`: Expo + React Native
- `shared`: shared contracts, schemas, API helpers, and utilities
